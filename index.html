xmbd

Cross-provider media embedding with jQuery

View project onGitHub

Introduction

Embedding Media into your shiny webpage can sometimes really be a pain in the ass, especially when it comes to using events or forcing the media-object to pause, play or stop.

So I've come up with a simple solution: Put all those methods, events and other things into a small library! This is xmbd.

But wait, there's more: xmbd not only gives you this abilities for - say - YouTube, no the things provided here are for everyone!

Currently supported Providers are:

  • YouTube
  • Vimeo
  • Dailymotion

and more to come!

Including a media-object is now as simple as $('#someid').xmbd().embed('vimeo', '75320274')

See the README for further details.

Install

You can install xmbd via bower:
bower install xmbd

Or just download the Code from GitHub. You can find the minified version of the source-code in build/xmbd.min.js

Dependencies are:

  • jQuery
  • swfobject (for flash-stuff)

Building

xmbd is build automatically by TravisCi

To build xmbd by yourself follow these commands:

npm install
npm install grunt-cli -g
npm test

A minified version of xmbd is now in the build-folder!
